# Accessibility Audit (a11y)

Use a ferramenta Glob para encontrar todos os arquivos `**/*.tsx` dentro de `src/`, excluindo `node_modules`. Em seguida, use a ferramenta Read para ler cada arquivo encontrado. Depois de ler todos os arquivos, analise o conteúdo em busca de problemas de acessibilidade conforme os critérios abaixo e reporte os problemas agrupados por categoria.

## O que verificar

### Imagens
- `<img>` sem atributo `alt` ou com `alt=""` em imagens não-decorativas
- Componentes `<Image>` (Next.js) sem `alt`

### Formulários
- `<input>`, `<select>`, `<textarea>` sem `<label>` associado (via `htmlFor`/`id` ou `aria-label` ou `aria-labelledby`)
- Botões de submit sem texto visível nem `aria-label`

### Botões e links
- `<button>` com conteúdo apenas visual (ícone) sem `aria-label`
- `<a>` com texto genérico ("clique aqui", "saiba mais") ou sem texto
- Elementos `<div>` ou `<span>` com `onClick` mas sem `role` e sem `tabIndex`

### Hierarquia de títulos
- Saltos de nível (`<h1>` seguido direto de `<h3>` sem `<h2>` no meio)
- Mais de um `<h1>` por página

### HTML semântico
- Uso de `<div>` ou `<span>` onde elemento semântico seria mais adequado (`<nav>`, `<main>`, `<section>`, `<article>`, `<header>`, `<footer>`, `<aside>`)
- Listas renderizadas sem `<ul>`/`<ol>` + `<li>`

### ARIA
- `aria-*` inválidos ou mal utilizados
- `role` customizado sem os atributos ARIA obrigatórios para aquele role

### Foco e teclado
- `tabIndex` com valor positivo (> 0), que bagunça a ordem de foco
- Uso de `outline: none` / `outline: 0` sem substituto de foco visível no CSS

## Formato do relatório

Para cada problema encontrado, informe:
- **Arquivo e linha** (link clicável)
- **Categoria** (Imagem, Formulário, Botão/Link, Títulos, Semântica, ARIA, Foco)
- **Nível WCAG** (A, AA ou AAA)
- **Descrição** do problema
- **Sugestão** de correção com trecho de código quando aplicável

Ao final, mostre um resumo: total de problemas por categoria e quantos são nível A (críticos), AA e AAA.

Se nenhum problema for encontrado em uma categoria, não a liste.
